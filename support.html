Ludus Support

Need help with Ludus? Want to report a bug or suggest a new feature? You can reach out to us through the following channels:

Official Support Servers

  • Psyvrse Discord - Main hub for all Ludus support, updates, and discussions.
  • The Bamboo Forest - Community server for Ludus users and bot enthusiasts.

Report Bugs or Issues

If you encounter any issues with Ludus, please:

  • Join one of our Discord support servers.
  • Provide a detailed description of the issue.
  • Include screenshots or error messages if possible.

Suggest Features

We love hearing ideas from our users! Suggestions can be submitted through the support servers. Make sure to describe:

  • The feature idea in detail.
  • How it would improve Ludus.
  • Any potential issues or concerns you foresee.
